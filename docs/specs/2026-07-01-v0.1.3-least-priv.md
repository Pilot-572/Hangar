# v0.1.3 — least-privilege role + install UX polish

Tight patch bump before the r/Proxmox launch post. Four small changes bundled.

## Motivation

Post-launch, the guaranteed top comment on r/Proxmox is *"why does it need Administrator?"* Fixing that pre-post kills the roast. Bundling three other papercuts because they're cheap.

## Scope

1. **HangarOps custom role** in `install.sh`. Replaces `Administrator` role grant.
   - Privs: `Sys.Audit, VM.Audit, VM.PowerMgmt, VM.Config.Options, VM.GuestAgent.Audit`
   - Maps 1:1 to what `app.py` actually calls:
     - `Sys.Audit` — `GET /cluster/resources` (node + VM listing, node CPU/mem)
     - `VM.Audit` — `GET /nodes/*/config` (tags, LXC IP)
     - `VM.PowerMgmt` — `POST /nodes/*/status/{start,shutdown,reboot}`
     - `VM.Config.Options` — `PUT /nodes/*/config` (tag edits)
     - `VM.GuestAgent.Audit` — `GET /nodes/*/qemu/*/agent/network-get-interfaces` (Proxmox 8.2+; initial spec listed `VM.Monitor`, which doesn't exist on modern Proxmox)
   - Idempotent both ways: creates or updates the role; grants HangarOps; removes any old Administrator ACL for `hangar@pam` if present. Re-running install.sh downgrades old installs cleanly.

2. **Startup-timeout diagnostic** in `install.sh`. Current behavior: after 30s wait loop, if `curl` never got a 200 but the LXC has an IP, the installer prints the URL and exits. User hits a dead page with no diagnostic. Fix: on timeout, run `pct exec CTID -- systemctl status hangar --no-pager` and print it before failing. +3 lines.

3. **README** update: change the Security callout from "Administrator role" to "custom HangarOps role, minimum privs" so the doc matches reality. Update the manual `pveum` quickstart block to use HangarOps too.

4. **CSS card alignment fix** already on `main` (commit 3816075). Tag included.

5. **install.sh REF** bumps to `v0.1.3`.

## Migration for existing (v0.1.0–v0.1.2) installs

One-liner in release notes:

```bash
pveum role add HangarOps -privs "Sys.Audit,VM.Audit,VM.PowerMgmt,VM.Config.Options,VM.GuestAgent.Audit" \
  && pveum acl modify / -user hangar@pam -role HangarOps \
  && pveum acl delete / -user hangar@pam -role Administrator
```
